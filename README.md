■プロジェクト概要\n
「ニュートンラプソン法を用いた逆運動学」

■シミュレータ動画

https://github.com/user-attachments/assets/717df68f-953f-4a63-8f34-a11da0c3764c

■開発の目的
反復計算により逆運動学を解き、ロボットシミュレータ CoppeliaSim を使用して反復の収束を視覚化した。

■詳細
・現在の関節角から順運動学を用いてエンドエフェクタの姿勢Tsbを計算する。
・目標姿勢Tとの誤差をツイストVbとして求める。
・ボディヤコビアンの擬似逆行列を用いて関節角を更新し、誤差が許容値以下になるまで反復計算を行う。
・各反復時の関節角は履歴として保存する。
・各反復時の関節角に基づき、ロボットシミュレータを操作することで、反復の収束を視覚化した。
